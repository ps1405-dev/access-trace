# Auralis

An audio-first, keyboard-operable science lab for blind and low-vision secondary-school learners. Built for OpenAI Build Week 2026, Education track.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## What works

- Circuit, motion, and wave labs with live controls
- Browser-native sound mappings using Web Audio
- Spoken explanations through the browser speech engine
- Keyboard-operable sliders and controls
- Deterministic calculations and concept-check feedback

No API key or paid service is required to run Auralis.

## Notes

Use a modern Chromium browser for the best Web Audio and speech-synthesis support. Start audio by pressing the visible play button; browsers intentionally require a user gesture before producing sound.
